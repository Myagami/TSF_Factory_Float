□戦術機製造産業セット
○概要
今回は当セットをダウンロードしていただき、まことにありがとうございます。
このセットはageが発売しているマブラヴ（アドオン的には主にオルタネイティヴ寄り）というゲームに登場する戦術機という物をSimutrans内で擬似的に製造する産業セットです。
産業としてはかなり設定がしょっぱいですのでご了承下さい
